# Tech Stack

## Backend
- **Language:** Java 11
- **Framework:** Spring Boot (2.5.14)
- **Data Access:** Spring Data JPA, Hibernate
- **Security:** Spring Security (OAuth2 Client, JOSE)
- **API Style:** Spring HATEOAS
- **Caching/Data Store:** Redis
- **Validation:** Spring Boot Starter Validation

## Frontend
- **Language:** JavaScript
- **Styling:** Bulma CSS, Sass
- **Library:** jQuery
- **Build Tool:** Webpack
- **Utilities:** Luxon, Clipboard.js, Fancybox

## Infrastructure & Tools
- **Build System:** Gradle
- **Documentation:** Asciidoctor
- **Continuous Integration:** Concourse
